# Workshop Archive — PRD

## 1. 서비스 개요
Workshop Archive는 셀피쉬클럽 워크숍 참여자들의 프로젝트를 한곳에 모아 서로 영감을 주는 커뮤니티형 포트폴리오 사이트다. 워크숍 참여자뿐 아니라 외부 방문자도 자유롭게 둘러볼 수 있으며, Product Hunt와 Behance의 감성을 결합한 카드형 갤러리로 프로젝트를 탐색할 수 있다. 비영리로 운영되며 셀피쉬클럽의 브랜딩과 커뮤니티 확장에 기여하는 것이 핵심 목표다.

## 2. 폴더 구조
이 레포는 5개 팀(파트)에 따른 5개 영역으로 나뉜다:
- planning/ — 기획 (무엇을 만들 것인가)
- design/ — 디자인 (어떻게 보여줄 것인가)
- cx/ — CX (누가 어떻게 사용할 것인가)
- marketing/ — 마케팅 (어떻게 알릴 것인가)
- strategy/ — 전략 (어떻게 지속 가능하게 만들 것인가)

각 영역의 상세는 해당 폴더에서 작성된다.

## 3. 기획 (Planning)
- 프로젝트 갤러리: 카드형 그리드로 전체 프로젝트를 탐색할 수 있는 메인 화면
- 좋아요/영감 표시: 다른 참여자의 프로젝트에 반응을 남겨 영감을 교환
- MVP 범위: 프로젝트 등록 + 갤러리 뷰 + 영감 반응 기능까지

상세는 `planning/` 폴더 참조. [작성 예정]

## 4. 디자인 (Design)
- 깔끔하고 영감을 주는 갤러리형 레이아웃. Behance/Dribbble의 비주얼 탐색 경험과 Notion 갤러리의 심플함을 결합
- 메인(갤러리) → 프로젝트 상세 → 영감 반응의 직관적 흐름

상세는 `design/` 폴더 참조. [작성 예정]

## 5. CX (Customer Experience)
- 핵심 페르소나: 셀피쉬클럽 워크숍을 수료하고 자신의 프로젝트를 공유하고 싶은 참여자, 그리고 워크숍에 관심 있는 잠재 참여자
- 핵심 여정: 사이트 방문 → 갤러리 탐색 → 프로젝트 상세 확인 → 영감 반응 남기기

상세는 `cx/` 폴더 참조. [작성 예정]

## 6. 마케팅 (Marketing)
- 워크숍 참여자에게 직접 안내하여 자연스럽게 첫 콘텐츠 확보. 참여자 본인이 공유하고 싶은 동기를 활용
- 주요 채널: 셀피쉬클럽 커뮤니티(카카오/슬랙), 인스타그램

상세는 `marketing/` 폴더 참조. [작성 예정]

## 7. 전략 (Strategy)
- 비영리로 운영하며, 셀피쉬클럽 브랜드 인지도와 워크숍 참여 전환율을 높이는 마케팅 자산으로 활용
- 아카이브 누적 → 커뮤니티 신뢰 → 신규 워크숍 참여자 유입의 선순환 구조

상세는 `strategy/` 폴더 참조. [작성 예정]

## 8. 역할 배정 (가상)
이 프로젝트를 팀으로 한다면:

| 팀 | 담당 영역 | 작업 폴더 |
|----|---------|----------|
| PM (본인) | 전체 기획·의사결정 | 루트 |
| 기획 팀 | 기능 정의, MVP 범위 | planning/ |
| 디자인 팀 | UI/UX, 비주얼 | design/ |
| CX 팀 | 페르소나, 여정맵, FAQ | cx/ |
| 마케팅 팀 | 런칭, 채널 | marketing/ |
| 전략 팀 | 수익 모델, 성장 | strategy/ |
