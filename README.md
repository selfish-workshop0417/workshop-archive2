# Workshop Archive

셀피쉬클럽 워크숍 참여자들의 프로젝트를 한곳에 모아 서로 영감을 주는 아카이브 사이트

## 프로젝트 정보
- **유형**: 커뮤니티 + 포트폴리오
- **타겟 사용자**: 워크숍 참여자 및 외부 방문자 누구나
- **출시 목표**: 워크숍 기간 내

## 핵심 기능
- 프로젝트 갤러리 (카드형 그리드로 프로젝트 탐색)
- 좋아요/영감 표시 (다른 참여자 프로젝트에 반응 남기기)

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
